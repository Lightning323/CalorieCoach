# Mongo DB

# APIs used

## Open food facts database (API)
https://github.com/openfoodfacts/openfoodfacts-js

No API key required for OpenFoodFacts

## Gemini API
https://ai.google.dev/gemini-api/docs/quickstart

Put the API key in .env under `GEMINI_API_KEY`
